
# E-Commerce Sales Analytics Dashboard

## Project Overview
Interactive sales analytics dashboard analyzing 99,000+ real e-commerce 
transactions from Olist, a Brazilian marketplace.

## Tools & Technologies
- **SQL (MySQL)** — data extraction, joins, aggregations
- **Power BI** — dashboard design, DAX measures, visualizations
- **DBeaver** — SQL client for query development

## Key Insights
- Total Revenue: R$15.84M across 99K orders
- Top Category: Health & Beauty drives highest revenue
- Avg Order Value: R$159.33
- 96K unique customers across all Brazilian states

## Dashboard Features
- KPI cards: Revenue, Orders, Avg Order Value, Customers
- Revenue trend by month (2016–2018)
- Top 10 product categories by revenue
- Revenue distribution by customer state
- Interactive slicers: date range and state filter

## Dataset
Olist Brazilian E-Commerce Dataset — kaggle.com/datasets/olistbr/brazilian-ecommerce
100,000+ real transactions from 2016–2018
